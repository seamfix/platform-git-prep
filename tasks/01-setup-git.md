# Task 01: Setup Git

## Objective
Install Git, configure user settings, and connect to GitHub.

## Steps
1. Install Git
2. Run `git config --global user.name "Your Name"`
3. Run `git config --global user.email "you@seamfix.com"`
4. Generate SSH key and add it to GitHub
5. Confirm setup with `git config --list`

## Deliverable
Comment in your pull request with a screenshot of your config list.
